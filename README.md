QualityCenterBrowser
====================

QualityCenter浏览器;主要用于浏览访问HP Quality Center.
本代码大量借鉴了WxPython2.9.X中的Html2_WebView实例

----------------------------------
支持WinXP/Win2003/Win7/Win8/Win2008等所有Windows操作系统
支持老版本的TD、QC以及最新版本的ALM

Requirements
-------------------------
### Platforms
- Python2.x
- wxpython2.9.x
